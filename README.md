# README

## Warehouse & Sales Flow System

A web-based system for managing products, warehouse stock, and the offer-to-order sales flow, designed for small and medium businesses in a multi-organization model. It supports item-level inventory tracking, stock operations (receipts, issues, adjustments), and versioned sets of offer and order line items. SaaS architecture based.

### Technologies
Ruby 3.3.6
Rails 8.1.1
PostgreSQL
